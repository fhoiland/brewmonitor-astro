# BrewMonitor Astro

Ren, statisk Astro-versjon av BrewMonitor.

## Stack

- Astro
- Tailwind CSS
- Astro Content Collections

## Kjør lokalt

```bash
npm install
npm run dev
```

## Innhold

- `src/pages/` for sider
- `src/content/blog/` for blogginnlegg
- `src/data/site.ts` for enkel mock-data og tekst

## Neste steg

- legge tilbake live bryggedata via liten serverless proxy
- eventuelt synkronisere blogg fra Notion
- deploye frontend til GitHub Pages
